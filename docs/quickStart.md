

### 安装

```shell 
从npm或者cnpm 安装
npm/cnpm install x-amuse -S
或者使用yarn
yarn add x-amuse
```

### 引入

```js
import Vue from 'vue'
import App from './App.vue';
import XAmuse from 'x-amuse'
import 'x-amuse/dist/css/styles.css'

Vue.use(XAmuse);

new Vue({
  el: '#app',
  render: h => h(App)
});
```
